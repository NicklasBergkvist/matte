# Matte Träning Åk 6

Detta projekt innehåller övningsmaterial i matematik anpassat för årskurs 6, baserat på områden som ofta tas upp i nationella prov.

## Innehåll

Materialet är uppdelat i flera HTML-sidor:

*   **Ordlista:** Förklaringar av vanliga mattebegrepp.
*   **Facit & Förklaringar:** Svar och genomgångar till övningsuppgifter.
*   **Prov:** Ett exempelprov.

(Observera: Övningsuppgifterna finns för närvarande endast i `frågor.md`)

## Live Version

Du kan se och använda materialet live via GitHub Pages på:

[https://nicklasbergkvist.github.io/matte/](https://nicklasbergkvist.github.io/matte/)

## Hur det byggs

Sidan publiceras automatiskt via GitHub Actions (`.github/workflows/static.yml`) när ändringar pushas till `main`-branchen.
